# Yadav Enterprises - Online Store 🏪

[![Live Demo](https://img.shields.io/badge/demo-live-brightgreen)](https://rpy2006.github.io/Yadav-Enterprises/)
[![Catalog Tool](https://img.shields.io/badge/tool-catalog-blue)](https://rpy2006.github.io/Yadav-Enterprises/catalog.html)

A lightweight, zero-backend e-commerce website for a local general store in **Mangan, North Sikkim**. Customers can browse products, add to cart, and place orders with COD or UPI payment.

## 🌐 Live Demos

| Link | Description |
|------|-------------|
| [**Main Store**](https://rpy2006.github.io/Yadav-Enterprises/) | The main e-commerce website - browse products, add to cart, and place orders |
| [**Catalog Generator**](https://rpy2006.github.io/Yadav-Enterprises/catalog.html) | Tool to generate product catalog code (for shopkeeper use) |

## ✨ Features

- **Product Catalog** - 24+ items across 8 categories (Groceries, Dairy, Snacks, Beverages, Stationery, Personal Care, Household)
- **Smart Search** - Real-time product filtering
- **Shopping Cart** - Persistent storage using localStorage
- **User Data Persistence** - Customer details auto-saved
- **Dark Mode** - Toggle between light/dark themes
- **Payment Options** - Cash on Delivery & UPI QR code
- **WhatsApp Integration** - Quick chat support
- **Catalog Generator Tool** - Easy product management for shopkeeper

## 🛠️ Built With

- HTML5
- CSS3 (Custom properties for theming)
- Vanilla JavaScript
- Font Awesome Icons
- QRCode.js library

## 🚀 How to Use

1. Visit the [main store](https://rpy2006.github.io/Yadav-Enterprises/)
2. Browse products by category or use search
3. Add items to cart
4. Proceed to checkout and enter details
5. Choose payment method (COD or UPI)
6. For UPI, scan QR code to pay
7. Order will be confirmed via WhatsApp

## 🛒 For Shopkeeper

Use the [Catalog Generator Tool](https://rpy2006.github.io/Yadav-Enterprises/catalog.html) to:
- Add/edit products
- Generate product code
- Update inventory
- Copy code to main website
